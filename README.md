# Hotel Review Dataset

Korean hotel review dataset for crawling practice in Text Mining Techniques course (2026 Spring).

400 synthesized hotel reviews across 4 pages (100 reviews per page).

## Columns
| Column | Description |
|--------|-------------|
| stars | Rating (1–5) |
| dates | Review date (year/month) |
| titles | Review title |
| contents | Review body |
| types | Travel type (커플/가족/친구/비지니스/단독) |
| photos | Photo attached (있음/없음) |
| grade | Hotel grade (4성급/5성급) |
| ranking | Hotel ranking (1–15) |

## Pages
- [Page 1](reviews_1.html): Reviews 1–100
- [Page 2](reviews_2.html): Reviews 101–200
- [Page 3](reviews_3.html): Reviews 201–300
- [Page 4](reviews_4.html): Reviews 301–400

## GitHub Pages Setup
1. Create a new public repository on GitHub
2. Upload all files in this folder
3. Go to Settings → Pages → Source: main / (root) → Save
4. Access via `https://{username}.github.io/{repo-name}/`
